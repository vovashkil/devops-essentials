CodeStar: Project Template

Introduction

Project templates are pre-existing solution stacks that include a development platform, an application type, and an AWS service on which the application will deployed. At the time of this recording, there are over 30 project templates provided by CodeStar and I fully expect that AWS will only add to the existing set of project templates.

The sample project template we're going to use is an Express.js web application running on AWS Elastic Beanstalk. In another sublesson and after you generate the pre-packaged solution using CodeStar, you'll copy the html files from devops-essentials GitHub repository to the CodeCommit repository generated by the CodeStar project template. Keep in mind that we could choose to use any of the CodeStar project templates to run the course website so the important point I will demonstrate is how you can make changes to the sample solutions provided by AWS.

Launch a solution from a CodeStar Project Template

  • To begin, choose the CodeStar service from the AWS console.
  • Choose the Express.js Web application project template that runs on Elastic Beanstalk.
  • On the Create project page, you will enter a unique Project name, leave the AWS CodeStar would like permission to administer AWS resources on your behalf. checkbox enabled, and choose Next. Note that you can chose to Edit Amazon EC2 Configuration if you’d like to modify things like your instance type, VPC, and Subnet. We’ll leave the defaults. You can see that CodeStar illustrates the deployment pipeline and tools that this project template will be using: CodeCommit for the Source, Elastic Beanstalk for the Deploy stage, and CloudWatch for monitoring.
  • Choose an existing key pair Choose an Amazon EC2 Key Pair window and select the checkbox to acknowledge that you have accsess to the private key pair.
  • You can choose Skip and wait for CodeStar to generate the entire solution. It takes a few minutes for it complete. You can see the percentage complete bar to monitor its progress. Once it's complete, go to the next sublesson that describes the CodeStar Dashboard.
