Course Summary

Summary

In this DevOps Essentials on AWS course you learned - -

  • Describe the principles and practices of DevOps on AWS.
  • Design solution architectures using AWS Developer and Management Tools (e.g. CloudFormation, CodePipeline, CodeCommit, CodeBuild, CodeDeploy, CodeStar, etc.).
  • Create a deployment pipeline for this course's web application on EC2 instances using AWS Developer Tools.
  • Create a deployment pipeline for this course's web application using OpsWorks and AWS Developer Tools.
  • Create a deployment pipeline for this course's web application using Elastic Beanstalk and AWS Developer Tools.
  • Create a deployment pipeline for this course's web application using EC2 Container Service and AWS Developer Tools.
  • Create a deployment pipeline for this course's web application using a serverless architecture and AWS Developer Tools.
  • Create a deployment pipeline for this course's web application using AWS CodeStar.
  • Describe an ideal DevOps state for a software application/service and team.

Delete AWS Resources

To delete the AWS resources used in this lesson -

  • Go to the CloudFormation console and select the checkbox next to the stack you created in this exercise, select Actions, then Delete Stack.
  • Remove any S3 buckets and objects created during the exercises.
