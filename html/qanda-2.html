How can you protect confidential information passed as a parameter into a CloudFormation stack?

  • Put the information in SSM Parameter Store
  • Encrypt with KMS and CloudFormation will automatically decrypt
  • Include the attribute NoEcho with a value of true
  • Provide the name of an object in S3 to read the value from

What is AWS CodeCommit?

  • Fully-managed service that automatically synchronize files between GitHub and AWS
  • Fully-managed source control service to host secure and highly scalable private Git repositories
  • A cloud-based IDE to deploying applications to AWS
  • A service that automatically commits files across all AWS services
