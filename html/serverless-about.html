Serverless: About

About Serverless

AWS Lambda lets you run code without provisioning or managing servers. You pay only for the compute time you consume - there is no charge when your code is not running.

When running applications and services on serverless infrastructure, a service like API Gateway provide the capability of make your service available to users.

To use these services, you can go to the console -

  • AWS Lambda
  • Amazon API Gateway

The rest of the section will demonstrate how to automate the infrastructure and workflows of a serverless application.
