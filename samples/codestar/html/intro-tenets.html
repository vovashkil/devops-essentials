Core Tenets

Core Tenets of this Video Course

  • Focus on deployment pipeline and pipeline architectures (i.e. not software architectures like Microservices, Functions as a Service, etc.)
  • Provide tangible, open source samples that use different AWS services (e.g. traditional EC2, CodeDeploy, ECS, OpsWorks, Elastic Beanstalk, Serverless, CodeStar, etc.)
  • Use a building block for more complex and custom use cases. Will and cannot account for every variation
  • Use only AWS tools
  • While we'll be using many AWS tools in our examples, the core tools that all samples will use are AWS CloudFormation and AWS CodePipeline
