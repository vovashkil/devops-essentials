CodeStar: Introduction

Learning Objectives

In this lesson you'll learn about AWS CodeStar. CodeStar is essentially an aggregation service that helps you easily configure and manage the full deployment stacks so that you can quickly develop, build, and deploy application on AWS. It uses many existing AWS services such as CodePipeline, CodeCommit, CodeBuild, CodeDeploy, IAM, and CloudWatch. It also includes a wiki, interfaces to development toos, and extensions to third-party tools.

By the end of this lesson you will be able to -

  • Launch an application using AWS CodeStar with an Elastic Beanstalk Project Template
  • View and modify Dashboard tiles that provide quick-glance project information
  • Setup your local environment to connect to an AWS CodeCommit repository for your project
  • Manage users for your project
  • View and modify your deployment pipeline in CodePipeline
  • Connect to a third-party tool via Extensions
  • Push changes to CodeCommit to deploy a different application in Elastic Beanstalk
