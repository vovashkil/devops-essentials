Serverless: Summary

Summary

In this lesson on creating and deploying AWS resources via OpsWorks you learned -

  • How to create a one-button solution to launch all AWS resources for building and deploying a static website
  • The major constructs of a CloudFormation template
  • The core AWS resources that make up the CloudFormation template for this solution
  • The basic AWS CodePipeline constructs and the constructs used in this static website
  • How to use the AWS CodeBuild service to run basic commands to deploy static S3 resources
  • How to setup your local environment for regularly committing code to a GitHub repository
